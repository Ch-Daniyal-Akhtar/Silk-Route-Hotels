
# Hotel Booking Management System (MERN)

A full-stack hotel booking platform built with the MERN stack, enabling users to search and book hotels while providing hotel owners with management and analytics tools.

---

## 🚀 Features

* Multi-role system (User, Hotel Owner, Admin)
* Hotel listing management (CRUD)
* Advanced search & filtering (price, location, ratings, facilities)
* Real-time booking system with history tracking
* Analytics dashboard (revenue, bookings, performance)
* Secure authentication using JWT
* Payment integration
* Image upload support
* Responsive UI

---

## 🛠 Tech Stack

**Frontend:** React, TypeScript, Tailwind CSS, React Query
**Backend:** Node.js, Express.js, MongoDB, Mongoose
**Services:** Stripe, Cloudinary

## ⚙️ Setup

```bash
cd hotel-booking

# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend
npm install
npm run dev
```
